# HiraokaHyperTools.SomeBlazorComponents

ASP.NET Core Blazor utility components by Hiraoka Hyper Tools.

## ShowLoader

```xml
<ShowLoader Task="_loaderTask">
	(After load)
</ShowLoader>
```

## ShowSaver

```xml
<ShowSaver Task="_saverTask">
	(Child content)
</ShowSaver>
```

## ShowModal

```xml
<ShowModal Title="" Show="_visible">
	<Body>
	</Body>
	<Footer>
	</Footer>
</ShowModal>
```
